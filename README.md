# About

Tickets is as client side app, that retrieve support tickets from Zendesk API.

# Specification

```ruby
XCode 8.2.1
Objective-C
Cocoapods 1.1.1
```

# Instalation 

Tickets is working with CocoaPods. After checkout the project run:

```ruby
git clone https://github.com/eltonmendes/Tickets.git
pod install
```

Enjoy! :D
